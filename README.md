# Про народження ідеї проєкту

Ідея цього проєкту з'явилася після перегляду результатів роботи
"ArUcoTUI: Набір програмних інструментів для прототипування матеріальних взаємодій на портативних плоскопанельних дисплеях за допомогою OpenCV":
- https://github.com/howieliang/arucotui

Було цікаво зробити fork та провести серію експериментів.
Але треба було:
- розміщувати панель дисплею на столі;
- налаштовувати досить складні жести під сценарій переміщення матеріальних об'єктів.

# Віртуальний тренер із гри у шахи на основі ArUco-маркерів із звуковими підказками

1) веб-камеру зафіксовано над поверхнею із шаховою дошкою
2) ArUco_маркери (розмір 12 мм з бібліотеки 4x4_50 ) розміщено зверху шахових фігур
3) для кожної фігури створено аудіо-файл із:
- загальним описом поведінки фігури;
- описом можливих варіантів переміщення фігури по дошці;
4) звукові підказки надаються:
- після наближення фігури до камери;
- наближення спеціального маркер-контролера до маркерів фігур;
5) для діагностики роботи програми візуалізуються шахова дошка, маркери фігур
