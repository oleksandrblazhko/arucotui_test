# Persona

Є файл ../server/marker.py, який описує монолітну архітектуру сервера для обробки ArUco-маркерів

Є каталог aruco_server, який описує модульну архітектуру:
- для обробки ArUco-маркерів з еквівалетними функціями модульної архітектури
- нову функцію для фільтрації (злагодження) даних опису маркерів

# Tasks

1) порівняй зміст файлу монолітної архітектури із файлами модульної архітектури
2) визнач, чи є властивості функцій монолітної архітектури, які було втрачено у функціях модульної архітектури
3) проаналій файли модульної архітектури на можливі помилки зв'язків між файлами

# Format

Результат аналізу збережи у файлі .tasks/reorg_analyze.md
