1) запропонувати розташовувати по чотири групи маркерів у чотирьох місцях розташування:

- center - у центрі перед камерою;
- left - зліва від камери із разворотом на 90 градусів вліво
- rigth - зправа від камери із разворотом на 90 градусів вправо

2) записувати характеристики маркерів впродовж 10–15 секунд.
3) для кожного маркеру обчислити:
std(tvec_x), std(tvec_y), std(tvec_z), std(rvec_x), std(rvec_y), std(rvec_z)
